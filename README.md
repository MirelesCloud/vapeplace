# Vapeplace CBD Website
### Gatsby v2

## Development
`git clone git@github.com:MirelesCloud/vapeplace.git`
`npm install`
`gatsby develop`

## Build
`gatsby build`

## Plugins
`gatsby-plugin-typography`
`gatsby-plugin-google-fonts`
`gatsby-transformer-sharp`
`gatsby-plugin-sharp`

`react-anchor-link-smooth-scroll`
